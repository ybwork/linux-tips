Команды.

	psql - доступ к командной строке

	exit - выход из командной строки

	sudo -i -u postgres - использование аккаунта на моём сервере

	\q - выход из аккаунта на моём сервере

	ALTER USER postgres PASSWORD ''; - сменить пароль для postgres

	\du - показыввает таблицу с пользователями

Если linux.

	Открыть файл /etc/postgresql/10/main/pg_hba.conf

	Убедиться, что есть строка local all postgres peer

	CREATE USER yetbetter WITH PASSWORD 'asdf1234'; - создаёт пользователя (если это linux, то нужно чтобы имя и пароль от компа совпадали)

	Выведет CREATE ROLE

	Ввести роль USER
